CONSULTAR DEPARTAMENTO

 
Esta opción permite consultar o modificar la lista de divisiones políticas del sistema, si conoce el código del departamento que va a consultar, sino lo conoce seleccione la opción buscar.
 
 
Para editar seleccione la opción con el icono de lápiz el departamento así puede editar el código o el nombre, así mismo tiene la opción de adicionar en caso de requerirlo ingresando el código y nombre que desee adicionar, para guardar debe dar clic en aceptar.
