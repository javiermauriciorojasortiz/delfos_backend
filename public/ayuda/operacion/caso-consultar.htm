CONSULTAR CASO

 
Esta opción le permite consultar un caso específico para ello debe digitar el número de documento del paciente, seleccionar la opción buscar y le mostrara la información del caso del paciente. Si desea generar un nuevo caso, puede digitar el número de documento agregar, seleccionar nuevo caso, se habilitara el formulario para diligenciar la informacion del paciente, debe digitar la información del responsable principal y seleccionar el boton aceptar.
 
Si desea editar el caso seleccione el icono del lápiz, le habilitara poder habilitar la información del paciente.
 
Recuerde que debe digitar la información para el responsable principal para que le permita guardar los cambios.
 
En diagnostico no le permitira hacer ediciones, pero tiene la opcion hostorial clinico si le permite realizar ediciones.
 
Para editar el historial clínico seleccione el icono del lápiz y le permitirá el estado, el diagnóstico del paciente.
 
Una vez termine consultar el caso del paciente y las modificaciones que realizo debe seleccionar la opción aceptar, si no desea guardar los cambios da clic en cancelar y tiene la opción de cerrar el caso.
