CONSULTAR USUARIO

 
Esta opción permite realizar la consulta de un usuario especifico seleccionado, un tipo de usuario, una vez lo selecciona, puede definir las fechas que desea consultar, el estado y seleccionar buscar, para que le muestre la información filtrada.
 
 
En estado se desplegará las siguientes opciones:
 
Una vez realice el filtro, le mostrara la información solicitada.
 
En caso de querer consultar todos los usuarios activados seleccionar buscar sin poner ningún filtro y le mostrara los usuarios registrados en el sistema y todos los estados en que se encuentre, para modificar un usuario seleccione el lápiz, para eliminarlo seleccione la opción papelera.
